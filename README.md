Данный проект является **ДЕМОНСТРАЦИОННЫМ** симулятором фишинговой страницы Telegram, созданным для:
- Обучения основам веб-безопасности
- Понимания механизмов фишинговых атак
- Тестирования собственных систем безопасности
- Исследовательских целей

## 🔧 Установка и запуск

### Локальный запуск

```bash
# 1. Клонировать репозиторий
git clone https://github.com/your-username/telegram-phishing.git
cd telegram-phishing
```

# 2. Установить зависимости
npm install

# 3. Запустить сервер
node server.js

# 4. Открыть в браузере
http://localhost:3000
Деплой на Render
https://render.com/images/deploy-to-render-button.svg

Залейте код на GitHub

Зарегистрируйтесь на render.com

Создайте новый Web Service

Настройте:

Build Command: npm install

Start Command: npm start
